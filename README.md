# Chat
## Descripción
Simple aplicación que permite escribir y borrar mensajes.
## Instrucciones
1. Escribir el mensaje en el espacio dado.
2. Apretar el botón Enviar comentario.
3. Para borrar, apretar el botón Eliminar.
## Lenguaje de programación
* Javascript ES6
* CSS3
* HTML5
## Librería
* Bootstrap 4
